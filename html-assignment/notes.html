List of HTML tags:

p
Defines a paragraph of text.
img
Inserts an image into the HTML document.
ul
Represents an unordered list, a list with bullet points.
ol
Represents an ordered list, a list with numbered items.
blockquote
Indicates that the enclosed text is an extended quotation.
